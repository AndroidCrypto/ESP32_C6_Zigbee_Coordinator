Explanation of the commands available on the Serial Monitor
interface:

Be sure that you use '115200 baud' speed and 'No Line Ending' option.
  
Command Explanation
on      switch all connected lights on
off     switch all connected lights off  
toggle  toggle all connected lights
list    refresh the display (important for a first access)
1on     switch device 1 light on
1off    switch device 1 light off
2on     switch device 2 light on
2off    switch device 2 light off
freset  factory reset, clears the stored boundings
open_network opens the Zigbee network for bounding new devices
help    prints the available commands
